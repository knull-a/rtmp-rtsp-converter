How to View RTSP Streams

Using VLC Player:

  1. Open VLC Player
  2. Go to Media > Open Network Stream
  3. Enter the RTSP URL (e.g., rtsp://localhost:8554/stream1)
  4. Click Play

Using FFplay:

ffplay rtsp://localhost:8554/stream1
